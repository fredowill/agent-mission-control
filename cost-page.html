Methodology

Data source — Claude Code conversation transcripts (.jsonl files stored locally by Claude Code). Each API call records exact token counts in the response.

Token counts — Exact. Taken directly from Anthropic's API response fields: input_tokens, output_tokens, cache_creation_input_tokens, cache_read_input_tokens.

Pricing — Published Anthropic rates. Opus 4.6: $15/$75 per MTok (in/out), $18.75/$1.875 cache (write/read). Sonnet 4.6: $3/$15, $3.75/$0.30. Haiku 4.5: $0.80/$4, $1.00/$0.08.

Deduplication — Streaming chunks from the same API call share identical cache token counts. Grouped by (model, cacheRead, cacheWrite); only the entry with the highest output_tokens per group is counted.

Scope — This project's transcripts only (phredomade working directory). Data starts from the earliest available transcript file.
